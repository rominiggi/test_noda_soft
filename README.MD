Решение тестового задания от NodaSoft.

Запуск теста:
1. Изменить параметры доступа к БД (config/config_db.php).
2. Из браузера обратиться к test.php.

Тест выполняет:

1. создается временная (TEMPORARY) таблица Users;
2. В созданную таблицу добавляется один пользователь;
3. В созданную таблицу добавляется пять пользователей;
4. выводятся пользоватеои старше 40 лет;
5. Выводятся пользователи с определенными именами
